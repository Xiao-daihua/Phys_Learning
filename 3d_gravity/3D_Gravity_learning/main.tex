\textbf{ 
3D Gravity Reading Project 2025-2026
}
\bigskip

我预计于2025年9月开始进行一个为期6个月左右的3D Gravity方向的Reading Project。我的主要目的是从最初始的地方开始逐步研究到更现代的3D Gravity的内容。我的计划是列出一个下面的主线大纲。但是会根据实际情况添加很多支线任务辅助我的阅读！下面是我的阅读主线计划：
\begin{itemize}
  \item 第一部分： Fundamentals 
    \begin{itemize}
      \item Geometric Quantization 理论学习 - 参考Blau的讲义
        \begin{itemize}
          \item 支线1: 辛几何基础学习
          \item 支线2: 复几何基础学习「参考Topology的书」
        \end{itemize}
      \item 2D Gravity学习
        \begin{itemize}
          \item Turiaci的2D Gravity综述「后半部分是重点」
        \end{itemize}
      \item Verlinde 原始经典论文阅读
        \begin{itemize}
          \item 「Verlinde以及数学人的经典论文」不需要很细
        \end{itemize}   
    \end{itemize}
  \item 第二部分： 近代3D Gravity的Preliminary阅读
    \begin{itemize}
      \item AdS3 gravity and random CFT Cotler的文章【学习3D Gravity基础方法】
    \end{itemize}
  \item 第三部分：Virasoro TQFT以及更多
    \begin{itemize}
      \item Virasoro TQFT 基础文章【Eberhardt的文章学习】
    \end{itemize}
\end{itemize}

对于Morphisms，有下面各种术语进行对比：
\imp{Morphisms的定义和翻译}{
    \itm{
        \pt{homeomorphism：是拓扑空间的映射，满足：连续的双射；并且有连续的逆映射}
        \pt{Isomorphism：也就是一个映射，我们只要求这个映射preserve一些structure（具体请取决于语境），也就是映射前后东西是“一类的”；同时这个映射需要有逆映射。}
        \pt{Automorphism：首先是一个iso，但是同时是自己到自己的映射。}
        \pt{Endomorphism：首先是一个morph，但并不一定有逆映射所以不是iso；但是是自己到自己的。}
        \pt{biholomorphism：指的两个复平面之间的映射，需要是双射；同时正和逆映射都是holomorphic的！}

    }
}
